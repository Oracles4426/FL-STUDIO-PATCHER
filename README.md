## 🚀 Installation and setup

Follow these steps to apply the patch to your machine. 

1. Close all instances of your music production software. The patch will fail if the program runs in the background.
2. Locate the file you downloaded. 
3. Right-click the file icon.
4. Select Run as administrator. Your system will prompt you for permission. Click Yes to allow the tool to make changes.
5. A window will appear on your screen. This interface communicates directly with the system registry. 
6. Click the Patch button located in the center of the window. 
7. Wait for the progress bar to finish. The tool will display a message when it completes the operation.
8. Click OK and close the tool.
